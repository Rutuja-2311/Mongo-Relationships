A Node.js and MongoDB project created to learn and practice database relationships using Mongoose.

This project demonstrates how relations work in MongoDB, including:

1.One-to-One Relation

2.One-to-Many Relation

3.Many-to-Many Relation

4.Referencing with ObjectId

5.Population using .populate()

🚀 Technologies Used

1.Node.js

2.Express.js

3.MongoDB

4.Mongoose
